# Mortgage-backed securities management system

*A mortgage-backed securities management system for filtering and packaging mortgages using a 2012 NJ database.* 

### Dependencies
Database: https://www.consumerfinance.gov/data-research/hmda/historic-data/
JDBC and PostgreSQL Connection

## Use flow:
User menu will appear upon running locally, giving filtering, packaging, rate calculation, etc options for the NJ database.
